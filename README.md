# Unity-DMX

Targets Enttec DMX USB Pro Mk2 + Clones

Download the Unity DMX package here: https://github.com/davivid/Unity-DMX/raw/master/DMX/DMX.unitypackage

Currently Supports:
- DMX Output
- 1 Universe / 512 Channels
- Retrieve Serial number of device
